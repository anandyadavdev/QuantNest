# 📈 QuantNest - Full-Stack Financial AI Dashboard

**QuantNest** ek modern AI-powered financial analytics platform hai jo users ko unke financial data ko visualize karne aur Groq AI ke through deep insights lene mein madad karta hai.

🚀 **Live Demo:** [https://quant-nest.vercel.app/](https://quant-nest.vercel.app/)

---

## ✨ Key Features

* **AI Insights:** Groq AI (Llama3) ka upyog karke financial trends aur patterns ka deep analysis.
* **Interactive Dashboards:** React aur Chart.js ka use karke real-time data visualization.
* **Secure Authentication:** JWT-based secure login aur registration system.
* **Modern UI:** Responsive design jo smart casual aur formal themes ko support karta hai.
* **MERN Stack:** MongoDB, Express, React, aur Node.js par adharit robust architecture.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite, Chart.js, Tailwind CSS.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB Atlas (Cloud Storage).
- **AI Integration:** Groq API (Llama3 model) financial logic ke liye.
- **Deployment:** Vercel (Frontend) aur Render (Backend).
